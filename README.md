# article-json-react-render [![Build Status](https://travis-ci.org/micnews/article-json-react-render.png?branch=master)](https://travis-ci.org/micnews/article-json-react-render)

Render article json as react components

## Installation

```sh
npm install article-json-react-render --save
```


## Tests

```sh
npm install
npm test
```
